Компоненты отображения текстовой информации

Таблица 1 - Компоненты ввода и отображения текстовой информации

Компонент

Станица

Описание

1

Label (метка)

Standard

Отображение текста, который не изменяется пользователем. Никакого оформления не предусмотрено, кроме цвета метки и текста. Основное свойство - Caption

2

SteticText (метка с бордюром )

Additional

Подобен компоненту Label, но обеспечивает возможность задания стиля бордюра. Основное свойство - Caption

3

Panel(панель)

Standard

Компонент является контейнером для группирования органов управления, но может использоваться и для отображения текста с возможностями объёмного оформления. Основное свойство - Caption

4

Edit(окно редактирования)

Standard

Отображение, ввод и редактирование однострочных текстов. Есть возможность оформления объёмного бордюра. Основное свойство - Text

5

MaskEdit(окно маскированного редактирования)

Additional

Используется для форматирования данных или символов в соответствии с шаблоном . . Основное свойство – Text и EditText

6

LabeledEdit

Additional

Отображение, ввод и редактирование однострочных текстов с отображением тектса подсказки. Основное свойство - Text

6

Memo(многострочное окно редактирования)

Standard

Отображение, ввод и редактирование многострочных текстов. Есть возможность оформления объёмного бордюра. Основное свойство - Lines

7

RichEdit(многострочное окно редактирования в формате RTF)

Additional

Компонент представляет собой окно редактирования в стиле Windows в обогащённом формате RTF, позволяющее производить выбор атрибутов шрифта, поиск текста и многое другое. Основное свойство - Lines

8

ListBox(окно списка)

Standard

Отображение стандартного окна списка Windows, позволяющее пользователю выбирать из него пункты. Основное свойство - Items

9

CheckListBox(список с индикаторами)

Additional

Компонент является комбинацией списка ListBox >  и индикаторов CheckBox

10

ComboBox(редактируемый список)

Standard

Объединяет функции ListBox  и Edit. Пользователь может либо ввести текст, либо выбрать из списка. Основное свойство – Items

11

SringGrid(таблица строк)

Additional

Отображение текстовой информации в таблице из строк и столбцов с возможностью перемещаться по строкам и столбцам и осуществлять выбор. Основное свойство - Cells

Преобразование строк:

StrToInt (AnsiString S) – преобразует строку S в целое число;

StrToFloat (AnsiString S) – преобразует строку S в вещественное число;

IntToStr ( int > I) – преобразование целого числа I в строку;

FloatToStr ( float > I) - преобразование вещественного числа I в строку.

Пример:

void __fastcall TForm1::Button1Click(TObject *Sender)

{ int a, b;

a = StrToInt (Edit1->Text);    //   ввод числа а

b = StrToInt (Edit2->Text);   //   ввод числа b

Edit3->Text =IntToStr (a+b);   // вывод суммы чисел

Edit1-> Clear (); // очистка редактора 1

Edit2-> Clear (); //   очистка редактора 2

}
