Справка

  1. Компонент Memo

Задание 3

  1. Создайте новый проект;
  2. Поместите на форму компонент Memo;
  3. Добавьте на форму кнопку «Добавить», по нажатию на которую на форме появляется компонент Edit и кнопка «Сохранить», при нажатии на которую текст из компонента Edit добавляется в Memo.
  4. Добавьте на форму кнопку «Удалить», по нажатию на которую на форме появляется компонент Edit для ввода номера удаляемой строки и кнопка «Ok», при нажатии на которую появляется сообщение с запросом на подтверждении удаления, а затем текст из строки компонента Memo удаляется.
  5. Добавьте на форму кнопку «Вставить». Вставка текста в соответствующую строку.
  6. *Добавьте на форму кнопку «Отмена», выполняющую отмену последнего действия.
