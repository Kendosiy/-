  1. Таблица атрибутов компонента

Компонент RichEdit

Страница – Win32l

В компоненте Memo формат (шрифт, его атрибуты, выравнивание) одинаков для всего текста.

Компонент RichEdit работает с текстом в обогащённом формате RTF. При желании изменить атрибуты вновь вводимого фрагмента текста вы можете задать свойство SelAttributes. Это свойство типа TtextAtributes, которое в свою очередь имеет подсвойства: Color (цвет), Name(имя шрифта), Size(размер), Style(стиль) и ряд других.

Например, введите на форму компонент RichEdit, FontDialog со страницы Dialog и кнопку Button, которая позволит пользователю менять атрибуты текста. В обработчик щелчка кнопки введите текст:

if FontDialog1. Execute then RichEdit1.SelAttributes.Assign(FontDialog1.Font);

RichEdit1.SetFocus;

Запустите приложение и вы увидите, что можете изменять шрифт выделенного текста. Если ничего не выделено, то установленные атрибуты влияют на атрибуты нового текста, вводимого начиная с текущей позиции курсора. Позиция курсора определяется свойством SelStart.

Свойство DefAttributes содержит атрибуты по умолчанию, они действуют до момента изменения атрибутов свойством SelAttributes.

За выравнивание, отступы и т.д. в пределах текущего абзаца отвечает свойство Paragraph типа TParaAtributes. Свойства этого типа приведены в таблице:

Alignment

Выравнивание текста. Может принимать значение taLeftJastify (влево), taCenter (по центру), taRightJastify(вправо)

FirstIndent

Число пикселей отступа красной строки.

Numbering

Управляет вставкой маркеров. Значения nsNone – отсутствие маркеров, nsBullet – маркеры ставятся.

LeftIndent

Отступ в пикселях от левого края

RightIndent

Отступ в пикселях от правого края

TabCount

Количество позиций табуляции

Tab

Значение позиций табуляции в пикселях

Значения свойства Paragraph можно задавать только в процессе выполнения приложения. Значения его подсвойств относятся к тому абзацу, в котором находится курсор.

Пример.

Пусть в вашем приложении есть окно редактирования Edit, в которое вводится имя сотрудника и есть кнопка, при нажатии на которую в окно RichEdit должна занестись шапка характеристики этого сотрудника, после чего пользователь может заполнять текст характеристики.

Обработчик события щелчка по кнопке может иметь вид:

RichEdit1.Clear; //очистка

RichEdit1.Paragraph.Alignment:=taCenter; //выравнивание по центру

RichEdit1.SelAttributes.Style:= [fsBold]; //установление жирного шрифта

RichEdit1.Lines.Add('ХАРАКТЕРИСТИКА');

RichEdit1.Lines.Add('Сотрудник'+edit1.Text);

RichEdit1.SelAttributes.Assign(RichEdit1.DefAttributes);//восстановление атрибутов по умолчанию

RichEdit1.Paragraph.Alignment:=taLeftJustify;

RichEdit1.SetFocus;

Загрузка в окно RichEdit1 текста из файла может осуществляться командой

RichEdit1.Lines.LoadFromFile(‘text.txt’);

Сохранение в файле:

RichEdit1.Lines.SaveToFile(‘text.txt’);
