Справка

  1. TRadioGroup, TRadioButton

Задание 3

Создайте проект, в котором задаётся массив, отображаемый в компоненте StringGrid.

Количество строк и количество столбцов задаётся с помощью счётчиков (UpDown или SpinButton), а способ задания его элементов (с клавиатуры или с помощью датчика случайных чисел) определяется с помощью компонентов:

А) RadioGroup;

Б) RadioButton.

При вводе элементов массива пользователем используйте функцию InputBox.

Выполните обработку массива (по вариантам). С помощью независимого переключателя организуйте выбор

· Отображения или нет на экране исходной таблицы;

· Отображения или нет на экране промежуточных результатов.

Вариант 1.

Дан вещественный массив А размером m×n. Определить k – количество «особых» элементов массива А и вывести их на экран с указанием местоположения, считая элемент особым, если он больше суммы остальных элементов его столбца.

Вариант 2.

Задан массив А размером n×n. Переставить строку с максимальным элементом на главной диагонали со строкой с заданным номером m.

Вариант 3.

Дана целая квадратная матрица n-ого порядка. Определить, является ли она магическим квадратом, то есть такой, в которой суммы элементов во всех строках и столбцах одинаковы.

Вариант 4.

Элемент массива называется седловой точкой, если он является наименьшим в своей строке и одновременно наибольшим в своём столбце или, наоборот, наибольшим в своей строке и одновременно наименьшим в своём столбце. Для заданной целой матрицы размером m×n напечатать все индексы её седловых точек

Вариант 5.

Дана вещественная матрица размером m×n. Переставляя её строки и столбцы, добиться того, чтобы наибольший элемент (или один из них) оказался в верхнем левом углу.

Вариант 6.

Определить, является ли заданная целая квадратная матрица n-ого порядка симметричной относительно главной диагонали.

Вариант 7.

Упорядочить по возрастанию элементы каждой строки матрицы размером m×n.

Вариант 8.

Задана матрица размером m×n. Найти максимальный по модулю элемент матрицы. Переставить строки и столбцы матрицы таким образом, чтобы максимальный по модулю элемент был расположен на пересечении k-ой строки и k-ого столбца.

Вариант 9.

Задана квадратная матрица. Получить транспонированную матрицу, то есть матрицу, где столбцы и строки меняются местами.

Вариант 10.

Составить программу, для нахождения матрицы - произведения двух квадратных матриц.

Вариант 11.

В данной действительной квадратной матрице порядка n найти наибольший по модулю элемент. Получить квадратную матрицу порядка n-1 путём отбрасывания из исходной матрицы строки и столбца, на пересечении которых расположен элемент с найденным значением.

Вариант 12.

Определить номера тех строк целочисленной матрицы А размером m×n, которые совпадают с одномерным массивом D, состоящим из n элементов. Если таких строк нет, то вывести соответствующее сообщение.

Вариант 13.

Расположить столбцы матрицы размером m×n в порядке возрастания элементов k-ой строки (1 ≤ k ≤ m).

Вариант 14.

Дана прямоугольная матрица. Найти строку с наибольшей и наименьшей суммой элементов. Вывести на печать найденные строки и суммы их элементов.

Вариант 15.

Квадратная матрица, симметричная относительно главной диагонали, задана верхним треугольником в виде одномерного массива. Восстановить исходную матрицу и напечатать по строкам.
