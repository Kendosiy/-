Диалоговые окна различных параметров

Windows поддерживает стандартные диалоговые окна, предназначенные для открытия и сохранения файлов, выбора и настройки шрифта, цвета, принтера и т.д. Для таких диалогов в D имеется специальный класс TCommonDialog. Компоненты для создания стандартных диалогов находятся в палитре компонентов на странице Dialogs. Это невизуальные компоненты, которые при выполнении программы будут не видны, поэтому их можно размещать в любое место на форме.

OpenDialog – диалог открытия файла. Он позволяет пользователю выбрать нужный файл и передает в основную программу имя выбранного файла, записанное в свойство FileName. Свойство Filter содержит фильтр, позволяющий выбирать файлы определенного типа, например текстовые. Для задания фильтра используется Редактор фильтра.

При использовании стандартного диалога важное место занимает метод Execute. Этот метод вызывает из основной программы стандартный диалог в модальном режиме. Если пользователь выбрал файл, то результатом работы метода будет True. Если пользователь нажал кнопку отмены, то результатом работы будет false.

Пример : загрузить в текстовый редактор Мемо содержимое текстового файла, который выбирается в стандартном диалоге открытия файла.

If OpenDialog1.Execute then Memo1.Lines.LoadFromFile(OpenDialog1.FileName) else Exit;

SaveDialog – диалог сохранения файла. В нем пользователь указывает имя и тип сохраняемого файла, а также путь для сохранения. Работа с ним аналогична работе с диалогом открытия файла.

FontDialog – диалоговое окно выбора шрифта. В нем пользователь выбирает шрифт и задает его характеристики. Основное свойство – Font – содержит вид и характеристики выбранного шрифта. В свойстве Device выбирается устройство, для которого задается шрифт (экран, принтер).

Пример: написать оператор, с помощью которого можно задавать параметры шрифта для Мемо с использованием стандартного диалога.

If FontDialog1.Execute then Memo1.Font:=FontDialog1.Font else Exit;

ColorDialog – диалог выбора цвета. Основное свойство – Color.
