Задание 2

Разработайте проект, в котором на главной форме помещено меню:

  • Данные:

o Ввод новой записи;

o Вывод

  • Поиск по параметру
  • О программе
  • Выход

Ввод значений полей записи осуществляйте во второй форме, вызываемой в модальном режиме.

Вывод данных на экран осуществляйте с помощью компонента StringGrid.

Результаты поиска отображайте с помощью StringGrid.

При щелчке по пункту меню «О программе» на экране должно появляться окно, созданное с помощью шаблона формы AboutBox, в котором содержится информация о разработчике.

Задание по вариантам:

1. Тип данных – Ученик (ФИО, дата рождения, адрес). Параметр поиска – дата рождения.

1. Тип данных – Спортсмен (ФИО, дата рождения, вид спорта). Параметр поиска – вид спорта.

2. Тип данных – Ученик (ФИО, оценки, средний балл). Параметр поиска – средний балл.

3. Тип данных – Рабочий (ФИО, зарплата, стаж). Параметр поиска – фамилия.

4. Тип данных – Конфеты (название, стоимость, срок годности). Параметр поиска – стоимость.

5. Тип данных – Ученик музыкальной школы (ФИО, инструмент, участие в конкурсах).

Параметр поиска – участие в конкурсах.

6. Тип данных – Рабочий (ФИО, зарплата, стаж). Параметр поиска – стаж.

7. Тип данных – Ученик (ФИО, группа, оценки). Параметр поиска – отсутствие «3» и «2».

8. Тип данных – Сотрудник (ФИО, возраст, стаж). Параметр поиска – возраст.

9. Тип данных – Преподаватель (ФИО, дисциплина, нагрузка). Параметр поиска – нагрузка.

10. Тип данных – Преподаватель (ФИО, дисциплина, нагрузка). Параметр поиска – дисциплина.

11. Тип данных – Ученик (ФИО, рост, вес). Параметр поиска – рост.

12. Тип данных – Продукт (название, стоимость, срок годности). Параметр поиска – срок годности.

13. Тип данных – Обувь (название, стоимость, размер). Параметр поиска – размер.
