  1. Главное и контекстное меню

Задание 1

Создайте проект, в котором на форме располагается главное меню следующей структуры:

  • Отобразить:
    • Кнопку
    • Метку
    • Окно редактирования
    • Фигуру
      • Круг
      • Прямоугольник
  • Скрыть
    • Кнопку
    • Метку
    • Окно редактирования
    • Фигуру
      • Круг
      • Прямоугольник
  • Выход
Напишите соответствующие процедуры для каждого пункта меню. Для компонента «Окно редактирования» создайте контекстное меню, с помощью которого это компонент можно увеличить, уменьшить, поменять цвет отображаемого окна редактирования.
