Задание 3

Добавьте в главное меню Вашего проекта пункты:

Файл

Создать

Открыть

Добавить

Закрыть

Разработайте процедуры, которые выполняются при выборе команд и пункта Файл:

Создать – создание нового файла,

Открыть – открытие существующего файла и отображение его содержимого в компоненте Memo или ListBox;

Добавить – добавление новой записи в файл с помощью функций InputBox;

Закрыть - Закрытие файла, открытого ранее, информация из которого отображается на экране.

Выход – выход из приложения.
