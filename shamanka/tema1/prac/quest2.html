Задание 2

Разработать программу для отображения типов данных языка Паскаль и их размеров (в байтах):

  1. Создайте новый проект (File|New Application)
  2. Выберите в инспекторе объектов свойство Caption (Заголовок) и вместо текста “Form1” введите текст “Типы данных”
  3. Из страницы Standard Палитры компонентов поместите на форму два компонента Label – метки, предназначенные для отображения текстовой информации
  4. Активизируйте Label1 и выбрав в инспекторе объектов свойство Caption (Заголовок), впишите его значение - “Integer”
  5. Активизируйте Label2 и выбрав в инспекторе объектов свойство Caption (Заголовок), впишите его значение - “Byte”
  6. Из страницы Standard Палитры компонентов поместите на форму два компонента Edit – окно редактирования, предназначенные для ввода и вывода текстовой информации
  7. Активизируйте Edit1 и Edit2 поочерёдно, выбрав в инспекторе объектов свойство Text (текст) очистите его значение
  8. Поместите на форму компонент Button (кнопка), выбрав в инспекторе объектов свойство Caption (Заголовок), впишите его значение - “Показать размер типов”
  9. Выровняйте и разместите на форме компоненты:
    • с помощью панели инструментов Align (выровнять), вызвав её командой View|Alignment Palette главного меню Delphi
    • с помощью изменений значений свойств Height (высота), Width (ширина), Left (расстояние от левого верхнего угла формы по горизонтали), Top (расстояние от левого верхнего угла формы по горизонтали). Значения задаются в пикселях.
    • c помощью мыши
  10. Измените цвет формы (свойство Color)
  11. Измените шрифт, которым написан заголовок на кнопке (свойство Font)
  12. Все используемые компоненты имеют имена (свойство Name). Измените имя компонента Edit1 на E1
  13. Двойным щелчком по кнопке Button1 откройте окно Редактора кода, обработчик события OnClick для кнопки Button1
  14. Наберите следующий код:
    E1.Text:=IntToStr(SizeOf(integer));
    Edit2.Text:=IntToStr(SizeOf(byte));
  15. Сохраните проект
  16. Запустите проект
  17. Доработайте проект таким образом, чтобы на форме отображалось название типа, комментарий на русском языке, размер типа в байтах для всех типов, используемых в языке Object Pascal:
    Integer, Cardinal, Shortint, Smallint, Longint, Int64, Byte, Word, Longword, Real, Real48, Single, Double, Extended, Comp, Currency, Char.
  18. Добавьте кнопку «Очистка», при нажатии на которую очищаются значения полей, в которых отображаются размеры
