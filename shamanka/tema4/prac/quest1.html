Справка

  1. Компонент StringGrid

Задание 1

Создайте проект, в котором ввод количества элементов массива задается с клавиатуры, а ввод элементов массива можно организовать или с клавиатуры или случайным образом. Вывод элементов массива осуществляется с помощью компонента StringGrid. Добавьте кнопку «Очистить», для очистки таблицы и «Выход» для закрытия приложения.

Выполните задание в соответствии с вариантом:

Дан одномерный целочисленный массив. Составить программы для решения следующих задач, используя

А) процедуры (функции) с циклами;

Б) рекурсивные процедуры (функции):

  1. Найти сумму модулей отрицательных элементов массива.
  2. Найти сумму квадратов элементов, больших некоторого числа X.
  3. Определить число соседств из двух чисел разного знака.
  4. Найти сумму элементов массива, являющихся точными квадратами.
  5. Найти количество элементов массива, являющихся двузначными числами.
  6. Найти наименьший положительный элемент массива.
  7. Найти количество четных элементов массива.
  8. Найти количество элементов массива, равных предпоследнему элементу.
  9. Найти первый отрицательный элемент массива.
  10. Найти наибольший трехзначный элемент массива.
  11. Найти наибольший элемент массива, имеющий ровно К делителей.
  12. Найти сумму элементов массива, имеющих ровно K цифр.
