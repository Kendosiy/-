Справка

  1. Панели общего назначения – компоненты Panel, GroupBox, Bevel, ScrollBox, Splitter.
  1. Создайте проект, аналогичный проекту приведенному в примере 1 .

На базе этого проекта выполните:

Задание 1:

  • Разработайте алгоритм решения задачи;
  • Разработайте тесты для проверки программы;
  • Составьте программу;
  • Выполните тестирование и отладку программы.

· На оценку «4» - одна программа, на оценку «5» - две программы.

Задание 2:

Вариант 1

Разработайте программу, заменяющую все гласные буквы текста соответствующими заглавными буквами. Используйте множественный тип при разработке программы.

Вариант 2

Определите число различных цифр в записи числа. Используйте множественный тип при разработке программы.

Вариант 3

Определите число различных букв в тексте. Используйте множественный тип при разработке программы.

Вариант 4

Удалите из текста слова, содержащие цифру. Используйте множественный тип при разработке программы.

Вариант 5

Определите число слов в тексте, содержащие специальные символы «@, #, $, ^, &, _, *, %, ~». Используйте множественный тип при разработке программы.

Вариант 6

Определите множество гласных букв, входящих в каждое слово текста. Используйте множественный тип при разработке программы.

Вариант 7

Определите множество гласных букв, входящих только в одно слово текста. Используйте множественный тип при разработке программы.

Вариант 8

Определите количество гласных букв, входящих в предложение. Используйте множественный тип при разработке программы.

Вариант 9

Сформируйте и выведите в алфавитном порядке все строчные буквы, встречающиеся в предложении. Используйте множественный тип при разработке программы.

Вариант 10

Подсчитайте количество знаков препинания в предложении. Используйте множественный тип при разработке программы.

Вариант 11

Удалите из предложения все цифры. Используйте множественный тип при разработке программы.

Вариант 12

Удалите из предложения все знаки препинания. Используйте множественный тип при разработке программы.

Вариант 13

В строке записано несколько чисел. Определите, какая цифра входит в запись каждого числа. Используйте множественный тип при разработке программы.

Вариант 14

Определите количество слов предложения, содержащих цифру. Используйте множественный тип при разработке программы.
